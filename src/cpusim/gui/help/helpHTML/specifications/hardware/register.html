Register

When you select "Register" in the popup menu at the top of the Edit Modules dialog box you will see a list of all existing registers. You can edit the parameters associated with any register or create new or delete old registers. The width of a register refers to the number of bits in the register. You cannot edit the contents of the registers (that is, the value stored in the registers) using this dialog box. To edit a register's contents, use the Register pane that appears on the main desktop.

Parameter:

Width: a positive integer that specifies the number of bits in the register. A register can be at most 64 bits wide.
